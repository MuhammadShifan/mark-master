# Mark Master

Welcome to **Mark Master**! This is a simple and efficient web application designed to help users calculate, track, and manage academic marks and grades easily.

### Live Demo
Check out the live project here: [Mark Master App](https://mark-master-mds.netlify.app/)

### Features
* **Grade Calculation:** Instantly compute total marks, averages, and final grades based on user input.
* **Interactive Dashboard:** Clean and intuitive interface for entering and viewing academic data.
* **Real-time Updates:** Dynamic logic ensures that results are displayed immediately as data is entered.
* **Responsive Design:** Optimized for seamless use on desktop, tablet, and mobile devices.

### Technologies Used
* **HTML5:** For the core structure of the application.
* **CSS3:** For styling, layout, and a visually appealing user interface.
* **JavaScript:** For form validation, mathematical calculations, and dynamic DOM updates.
